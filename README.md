## ðŸš€ installing

For install from source, first you need:

- clone repository
```sh
git clone https://github.com/roqueando/league-of-data.git
```

- create venv
```sh
python -m venv venv
```

- install dependencies
```sh
# activate virtualenv
source ./.venv/bin/activate

# install deps
pip install -r requirements.txt
```

- install hive jars
```sh
./setup.sh
```

- create the app image
```sh
make build
```
